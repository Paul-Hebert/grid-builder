# Grid Options

---

## Styles
* Number of columns
* Optional max-width
* Gutter width
    * Optional Outer Gutter
* Units

### Breakpoints
* Unlimited number
* Name
* number value

---

## Languages
* CSS
* SCSS
* SASS
* LESS
* Stylus

---

## Naming Convention
* col-{breakpoint-name}-{number}
* columns {breakpoint-name}-{number}

## Technical 
* Box sizing
    - Border Box
    - Content Box
* Semantic mix-ins
    - Only available when using a preprocessor
* Strategies
    - Flexbox
    - CSS Grid
    - Floats
    - Inline Block
* Include push/pull/offset classes
